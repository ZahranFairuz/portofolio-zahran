Web Developer

Saya adalah seorang Web developer dan saya besekolah di SMK Wikrama Bogor dengan pengalaman dalam membangun aplikasi dan situs web yang fungsional, responsif, dan user-friendly. Menguasai HTML, CSS, dan JavaScript, saya merancang antarmuka yang bersih dan interaktif, serta mengembangkan backend menggunakan Node.js dan Laravel. Selain itu, saya memiliki keterampilan dalam mengelola database dan menggunakan Git untuk version control, memastikan kode terorganisir dengan baik dalam proyek tim. Saya berkomitmen untuk menciptakan solusi web yang efisien dan mudah digunakan. Saya terus mengembangkan keterampilan saya untuk menciptakan aplikasi web yang efektif dan responsif.
